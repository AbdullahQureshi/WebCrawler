WebCrawler
==========

This is an android application which does web crawl for hyperlinks. The app stops crawling when 1000 links are gathered without user intervention.

There is a button which is labeled “begin” if crawling not in progress else “stop” to force stop crawling before 1000 links are gathered. 
Application utilizes up to 10 threads for crawling, It is capable of crawling gzip encoded pages and supports https connection.
